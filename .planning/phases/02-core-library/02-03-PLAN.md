   ```rust
   pub fn validate_currency_code(code: &str) -> Result<(), ValidationError>
   ```
   - Check code.len() == 3, return InvalidCurrencyLength if not
   - Check code.bytes().all(|b| b.is_ascii_uppercase()), return InvalidCurrencyFormat if not

   - Sign mismatch: units=100, nanos=-50 invalid
